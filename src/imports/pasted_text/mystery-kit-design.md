Design a modern, mobile-first e-commerce web app called "Unboxd" that sells random sports shirts.

CORE CONCEPT:
Users purchase a "mystery shirt" without knowing which team it is. They can customize exclusions before purchase.

STYLE:

* Dark mode with premium sporty feel
* Clean UI, similar to Nike / Adidas aesthetic
* Bold typography, minimal but dynamic
* Use card-based layouts and smooth animations
* Emphasize excitement, randomness, and exclusivity

PAGES TO DESIGN:

1. HOME PAGE

* Hero section: “Get Your Mystery Shirt”
* CTA button: “Order Now”
* Explanation of concept (3 steps: Choose size → Set exclusions → Receive shirt)
* Testimonials / social proof
* Featured shirts preview (blurred or hidden for mystery effect)

2. ORDER FLOW
   Step 1: Select Size (Required)
   Step 2: Exclusions (Optional)

* Avoid clubs (multi-select)
* Avoid leagues (dropdown/multi-select)
* Avoid colors (color picker)
  Step 3: Review & Checkout
* Show “Mystery Shirt Box” instead of product image
* Price + shipping
* Payment button

3. USER ACCOUNT DASHBOARD

* Profile info
* Order history
* Closet (main feature)

4. USER CLOSET PAGE

* Grid of owned shirts
* Each shirt card includes:

  * Shirt image
  * Team name (revealed after delivery)
  * Status (Owned / Trading / Sold)
* Button: “Add to Trade”

5. TRADING MARKETPLACE

* List of shirts available for trade
* Filters: size, team, league
* Each listing shows:

  * Shirt image
  * Owner
  * Trade options:

    * Trade with another shirt
    * Offer money
    * Request “trade with platform” (exchange for mystery shirt)

6. TRADE FLOW SYSTEM

* Trade request page:

  * Select shirt from your closet
  * Choose trade type:

    * Shirt-for-shirt
    * Shirt + money
    * Sell for money
* Trade status tracking:

  * Pending
  * Accepted
  * Shipped
  * Completed
  * Rejected
* Visual progress tracker

7. TRANSACTION HISTORY

* List of all purchases and trades
* Payment status
* Refunds (if any)

FEATURES TO INCLUDE IN UI:

* Notification system (trade updates, order updates)
* Gamification elements (badges like “Collector”, “Trader”)
* “Duplicate detection” label in closet
* Wishlist / liked shirts

COMPONENTS:

* Reusable cards for shirts
* Trade status badges
* Buttons (primary, secondary, danger)
* Filters and dropdowns

EXTRA TOUCHES:

* Add excitement elements like “You might get…” blurred previews
* Confetti or reveal animation when shirt is delivered
* Clean onboarding flow for new users

OUTPUT:
Design full responsive UI (mobile + desktop) with clear user flow and reusable components.
